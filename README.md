# IoT
Este repositorio recoje todo el proyecto realizado como parte de la asignatura de Fundamentos de Internet de las Cosas. Actualizaciones pendientes

Este trabajo combina tanto software como hardware en la implementación y despliegue de diversas funcionalidades por medio de una RaspberrPi 3, haciendo uso de protocolos de comunicación como I2C y SPI en componentes como un LED, un pulsador, un servomotor y un motor de corriente continúa, un sensor de temperatura, una pantalla LCD, un lector de tarjetas NFC y un microcontrolador L293D.
